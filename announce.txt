MultiMail is a free, open source (GPL) offline mail packet reader for
Unix/Linux, MS-DOS, OS/2, Windows and Mac OS X, using a curses-based
interface. It supports the Blue Wave, QWK, OMEN, SOUP and OPX formats.

The MultiMail home page is:

 http://multimail.sourceforge.net/


What's New in version 0.47?
---------------------------

 * Added support for Microsoft Visual C++, MinGW, and Open Watcom
 * OS/2 port now built with Watcom
 * Win32 port now built with MSVC
 * Starting and MMAIL directories now searched for archivers and editors
 * Bundled zip and unzip with Win32 port
 * Support for "GreekQWK"
 * Other minor stuff
